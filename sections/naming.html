Naming

To help with consistencies in naming conventions below are how things should be named for Liferay themes.

Web Content

  • {THEME_SHORT_NAME} - 4 - 6 Characters Long / no-spaces>
    MODERN INDIGO = MIND, LITHE = LITHE
  • Name: {THEME_SHORT_NAME} - Web Content Name # + (Embeded)
    ex. "LITHE - Featured Content 1" , "LITHE - Featured Content 2"
    If webcontent is embeded in theme add "(Embeded)"

Structure

  • Name: {THEME_SHORT_NAME} - Structure Name + {Structure}
    ex. "LITHE - Featured Content Structure"

Template

  • Name: {THEME_SHORT_NAME} - Template Name + {Template}
    ex. "LITHE - Featured Content Template"

Layout

  • starting from top to bottom from left to right.
  • "1-2-3-columns" display name "1-2-3 Columns Layout"
  • "2-1-2-columns" display name "2-1-2 Columns Layout"

_CSS

  • Class / Ids
    "lowercased and use "-" for spaces"
